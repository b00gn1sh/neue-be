# Contributing to neue-be

Thanks for your interest in contributing!

## Reporting Bugs

Please open an issue with:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser and OS
- Screenshots if relevant

## Feature Requests

Open an issue with:

- Clear description of the feature
- Why it would be useful
- How it fits the Swiss design philosophy (constraints > options)

## Questions?

Open an issue or start a discussion.
